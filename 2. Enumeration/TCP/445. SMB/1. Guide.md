# Default Commands 

List shares

```
smbmap -H 10.10.10.100
```

View share 

```
smbget -R -U "" smb://10.10.10.100/Replication
```

Mount shares 

```
sudo mount -t cifs  -o rw,guest,vers=3.0 //10.10.10.219/kanban/ /mnt/kanban
```
