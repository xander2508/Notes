
## What is Android?

Modified version of the Linux kernel and other open source software

### Key Components

1. Android Open Source Project
	- Core of OS
	- Linux Kernel
	- Android runtime
	- Core libraries
1. Android Framework
	- Software layer on top of AOSP
	- [[Activities]]
	- [[Services]] 
	- [[Broadcast Receivers]]
	- [[Content Providers]]
1. Android Apps
	- Software apps which run on top of the Android operating system
2. Android Services
	- Apps created and managed by the Android platform
	- Background services
		- Download manager 
		- Media player service 
		- Location service
